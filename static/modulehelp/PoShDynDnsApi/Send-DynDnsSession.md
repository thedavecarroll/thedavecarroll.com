# Send-DynDnsSession

## SYNOPSIS
The `Send-DynDnsSession` command extends the current session with the Dyn Managed DNS REST API.

## SYNTAX

```
Send-DynDnsSession [<CommonParameters>]
```

## DESCRIPTION
The `Send-DynDnsSession` command extends the current session with the Dyn Managed DNS REST API.

## EXAMPLES

### Example
```powershell
PS C:\> Send-DynDnsSession
```

Extends the current session.
